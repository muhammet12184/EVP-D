UYGULAMA ÇÖKÜYOR - ÇÖZÜM ADIMLARI

Düzeltmeler Yapıldı:
===================

1. Try-catch blokları eklendi
2. mounted kontrolleri eklendi  
3. İzin kontrolü geciktirildi (500ms)
4. Bluetooth tarama try-catch ile korundu
5. minSdkVersion 23 yapıldı
6. ProGuard kuralları eklendi

Şimdi Dene:
===========

1. Uygulamayı sil:
   - Telefondan uygulamayı tamamen kaldır

2. Yeniden kur:
   flutter pub get
   flutter run

3. İzinleri ver:
   - Bluetooth izni VER
   - Konum izni VER
   - Yakın cihazlar izni VER

4. Bluetooth'u aç:
   - Telefonun Bluetooth'u açık olmalı

HALA ÇÖKERSE:
=============

Logları göster:
flutter run --verbose

Ya da:
adb logcat | grep -i flutter

Hata mesajını buraya yaz.

MUHTEMEL SORUNLAR:
==================

1. Bluetooth kapalı
2. İzinler verilmemiş
3. Android 12+ için izin sorunu
4. Flutter Blue Plus başlatma hatası

Hepsi düzeltildi. Şimdi çalışmalı.
