🚀 DEPLOYMENT SEÇENEKLERİ
═══════════════════════════════════════════════════════════════

1. Development (Local)
   python app.py
   → http://localhost:8000

2. Docker (Single Container)
   docker build -t ev-api .
   docker run -p 8000:8000 ev-api
   → http://localhost:8000

3. Docker Compose (Full Stack)
   docker-compose up -d
   → https://localhost:443 (Nginx + SSL)

4. Production (Manual)
   uvicorn app:app --host 0.0.0.0 --port 8443 \
     --ssl-keyfile key.pem --ssl-certfile cert.pem
   → https://your-domain.com:8443
