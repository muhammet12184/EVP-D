# 🚗 YENİ NESİL MOBİLİTE VE YAŞAM EKOSİSTEMİ (SUPER APP)

**Vizyon:** Sadece bir araç uygulaması değil; finans, asistan, ev ve sosyal yaşamı birleştiren dijital bir yol arkadaşı.

---

## 🎯 X-FACTOR ÖZELLİKLER

### 🎭 Seçilebilir AI Karakteri (Persona)
Kullanıcı asistanını seçer:
- **Sadık Kâhya** (Kibar/Resmi)
- **Eğlenceli Kanka** (Samimi/Şakacı)
- **Sert Koç** (Disiplinli)

### 🤝 "İmece" Modu (Topluluk Yardımlaşması)
Yolda kalan kullanıcılar için yardım çağrısı sistemi. Yakındaki üyeler yardıma giderse yüksek ödül/puan kazanır.

## 🏗️ PROJE YAPISI

```
super-app/
├── mobile/              # Flutter mobil uygulama
├── backend/             # Go backend servisleri
├── ai-services/         # Python AI/ML servisleri
├── aws-infra/           # AWS bulut konfigürasyonları
├── obd-parser/          # OBD-II veri parser
└── docs/                # Dokümantasyon
```

## 🛠️ TEKNOLOJİ YIĞINI

- **Frontend:** Flutter (iOS & Android)
- **Backend:** Go (Golang)
- **AI/ML:** Python (TensorFlow, PyTorch)
- **Bulut:** AWS (IoT Core, Lambda, EC2, RDS)
- **Harita:** Mapbox
- **Blockchain:** Batarya pasaportu için sertifikasyon

---

## 🚀 HIZLI BAŞLANGIÇ

### Gereksinimler
- Flutter SDK (3.0+)
- Go 1.21+
- Python 3.11+
- AWS CLI
- Docker & Docker Compose

### Kurulum

```bash
# Flutter uygulaması
cd mobile
flutter pub get
flutter run

# Go backend
cd backend
go mod download
go run cmd/server/main.go

# Python AI servisleri
cd ai-services
pip install -r requirements.txt
python app.py
```

## 📦 MODÜLLER

### A. Yapay Zeka ve Kişiselleştirme
- Duygu analizi (ses ve yüz ifadesi)
- Bağlamsal zeka ve proaktif öneriler
- Akıllı öğrenme (rotalar, alışkanlıklar)

### B. Elektrikli Araç (EV) Özellikleri
- Plug & Charge (otomatik ödeme)
- Gerçekçi menzil tahmini (%99 doğruluk)
- Batarya pasaportu (Blockchain sertifikasyon)
- V2L/V2H (Vehicle-to-Load/Home)

### C. Benzinli/Dizel Araç Özellikleri
- OBD-II bağlantısı (tak-çalıştır)
- Akıllı yakıt asistanı
- Mobil ödeme
- AI tamirci (arıza yorumlama)
- EV simülasyonu (tasarruf raporu)

### D. Finans ve Yaşam (Super Wallet)
- Tek cüzdan (sigorta, MTV, HGS, şarj, yakıt)
- P2P araç kiralama
- Akıllı ev entegrasyonu

### E. Oyunlaştırma
- Sürüş ligleri (Eco-Coin ödülleri)
- Şehir kaşifi (sesli rehber)
